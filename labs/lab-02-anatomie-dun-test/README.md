# Lab 02 — Anatomie d'un test

## Objectifs

Maîtriser la structure Arrange-Act-Assert (AAA) et implementer des classes testables avec des cas limites.

## Exercices

### Exercice 1 : Calculator
Implementez une classe `Calculator` avec les méthodes `add`, `subtract`, `multiply`, `divide` et un historique des operations.

### Exercice 2 : StringUtils (AAA)
Ecrivez des tests structures AAA pour une classe `StringUtils` avec `capitalize`, `slugify`, `truncate`.

### Exercice 3 : TemperatureConverter
Implementez un convertisseur de temperatures (celsius/fahrenheit/kelvin) avec gestion des cas limites.

### Exercice 4 : DateRange
Implementez une classe `DateRange` avec `contains`, `overlaps`, `duration` et tests de bornes.

### Exercice 5 : PasswordValidator
Implementez un validateur de mot de passe (longueur, majuscule, chiffre, caractere special).

### Exercice 6 : PaginationHelper
Implementez un helper de pagination (`totalPages`, `currentItems`, `isFirstPage`, `isLastPage`).
